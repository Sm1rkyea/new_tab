# New Tab

A custom new tab page built with TanStack React.

## About

New Tab "replaces" your browser's default new tab page with a clean, minimal interface featuring a clock, temperature, search, and quick links (that are currently not editable).

## Built with

- [TanStack](https://tanstack.com)
- [React](https://react.dev)
- [TailwindCSS v4](https://tailwindcss.com)
- [Vite](https://vitejs.dev)
